--Overview--

The ERP Management System is a desktop-based application developed in Java Swing that simulates core functionalities of an Enterprise Resource Planning system.
It provides an integrated solution for managing Inventory, Sales, and Purchases through a secure and centralized interface.
This project demonstrates practical implementation of Object-Oriented Programming (OOP) principles, GUI development, and modular software design, reflecting real-world ERP workflows.

--Features--
-Authentication-
Secure login system
Controlled access to system modules

-Inventory Management-
Add, update, and delete inventory items
Tabular display of products
Centralized inventory control

-Sales Management-
Record sales transactions
Automatic bill total calculation
Maintain sales history

-Purchase Management-
Supplier management
Purchase invoice generation
Requisition tracking

-User Interface-
Central dashboard for navigation
Separate modules for each operation
Intuitive and user-friendly GUI

--Technologies Used--
Java
Java Swing
Object-Oriented Programming (OOP)
File Handling
Event Handling & JTable
